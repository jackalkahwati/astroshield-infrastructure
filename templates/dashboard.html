Available Endpoints

System Endpoints

  • GET /api API root endpoint - Overview and available endpoints
  • GET /api/health Health check endpoint

Spacecraft Management

  • POST /api/spacecraft Update spacecraft data
  • GET /api/spacecraft/{id}/status Get spacecraft status

Threat Detection & Analysis

  • GET /api/spacecraft/{id}/threats Get all spacecraft threats
  • POST /api/spacecraft/{id}/threats Add threat information
  • GET /api/spacecraft/{id}/cyber-threats Get cyber threats for spacecraft
  • POST /api/spacecraft/{id}/cyber-countermeasures Deploy cyber countermeasures

Spacecraft Analysis

  • GET /api/spacecraft/{id}/maneuvers Get spacecraft maneuvers
  • GET /api/spacecraft/{id}/intent Analyze spacecraft intent
  • GET /api/spacecraft/{id}/ccdm/{other_id} Analyze conjunction between two spacecraft
